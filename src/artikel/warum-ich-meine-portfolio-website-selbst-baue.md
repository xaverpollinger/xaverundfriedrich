Es gibt einfachere Wege, eine Portfolio-Website zu bauen. Squarespace, Webflow, ein fertiges Theme — fertig in einem Nachmittag.

Ich habe mich dagegen entschieden. Nicht aus Prinzip, sondern weil ich verstehen will wie die Dinge funktionieren, die ich täglich gestalte.

## Was „Zero-Dependency" bedeutet

Kein React, kein Tailwind, kein jQuery. Jede Zeile Code die ausgeliefert wird, habe ich selbst geschrieben oder bewusst eingebunden. Das bedeutet: ich kann jede Entscheidung begründen.

## Was ich dabei lerne

Fluid Typography via `clamp()`, CSS Custom Properties als Design-System, semantisches HTML das auch für Screenreader funktioniert. Dinge die ich als Designer kannte, aber nie wirklich verstanden hatte.
